# AI Automation Workflow

## Mục tiêu
Mô phỏng workflow tự động hóa: phân loại yêu cầu → checklist xử lý → phản hồi mẫu → human approval.

## Chạy
```bash
python automation_workflow.py
```

## Điều cần giải thích trong phỏng vấn
- Input/output contract
- Vì sao cần human approval
- Cách đo chất lượng phân loại
