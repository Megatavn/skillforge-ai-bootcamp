# SkillForge AI v1.0.0 — Portfolio Release

First GitHub portfolio release of SkillForge AI.

## Highlights

- Vietnamese-first mobile learning UI
- Offline-first architecture with LocalStorage
- 100+ AI Engineering micro-lessons
- Code Missions with runnable Python starter kits
- Evidence Vault for GitHub/CV proof
- Capstone 2.0: offline Document Q&A Assistant
- Career Tracks and Job Readiness Score
- Portfolio Export and Interview Defense workflows
- Android APK build support via Capacitor

## Recommended release asset

Attach the Android debug APK as:
