# GitHub Upload Guide — SkillForge AI

Recommended repository name:

```text
skillforge-ai-career-bootcamp
```

Recommended repository description:

```text
Offline-first AI Engineering career bootcamp app for Vietnamese beginners, featuring code missions, Evidence Vault, Capstone 2.0, portfolio export and interview defense.
```

## 1. Prepare project

```bash
cd skillforge-ai-career-bootcamp
npm ci
npm run build
```

## 2. Initialize Git

```bash
git init
git add .
git commit -m "Initial SkillForge AI portfolio release"
```

## 3. Create GitHub repo

Create a new public repository named:

```text
skillforge-ai-career-bootcamp
```

Then connect:

```bash
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/skillforge-ai-career-bootcamp.git
git push -u origin main
```
