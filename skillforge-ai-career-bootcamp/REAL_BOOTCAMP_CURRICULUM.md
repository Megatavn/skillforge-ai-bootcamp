Project: **AI Career Capstone: Document Q&A Assistant**

Core workflow:

```text
Local documents → retrieval → grounded answer → citation → evaluation → report
```

- 10 test questions
- expected source
- no-answer test
- rubric 100 points
- evaluation report
- limitation and next-step plan

## Job-Ready Gate

Recommended minimum before applying:

- Job Readiness Score ≥ 76%
- At least 3 project proof sets
- Capstone rubric ≥ 75
- At least 1 GitHub repo with README + screenshot
- Interview defense saved for capstone
